## 🎯 Overview

**Potato Disease Detection System** is a production-ready web application that uses a **hybrid CNN-BiLSTM deep learning model** to detect potato diseases from leaf images.

### 🌟 Key Highlights

| Feature | Description |
|---------|-------------|
| **CNN-BiLSTM Model** | Hybrid architecture with 97% accuracy |
| **3 Disease Classes** | Healthy, Early Blight, Late Blight |
| **Web Interface** | Professional glassmorphism UI |
| **Authentication** | Secure login/register system |
| **Real-time Detection** | Instant disease prediction |
| **Dashboard** | Interactive charts & statistics |

### 🧬 Why CNN-BiLSTM?

- **CNN**: Extracts spatial features from images
- **BiLSTM**: Captures sequential patterns and context
- **Attention Mechanism**: Focuses on disease-relevant regions
